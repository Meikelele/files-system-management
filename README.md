# Project Goal
The goal of this project is to build a web application for managing a file system structure based on metadata.

The system allows creating and managing:
- files,
- folders,
- folder hierarchy,
- file statistics and reports.

---

# Project Scope

The application will support:
- creating and managing files,
- creating and managing folders,
- assigning files to folders,
- building nested folder structures,
- generating statistics and reports based on stored data.
